## 🚀 Projects

### 🗃️ [StockFlow API](https://github.com/softwacar/StockFlowAPI)
> Inventory & order management REST API built with clean architecture

- **4-layer Clean Architecture** with Repository Pattern
- JWT role-based authentication (Admin/User), BCrypt hashing, global exception middleware
- Stock validation on order creation; cancellation auto-restores inventory
- Fully unit tested with **xUnit + Moq**

`ASP.NET Core 8` `EF Core` `SQL Server` `JWT` `xUnit` `Moq`

---

### 💬 [ChatFlow API](https://github.com/softwacar/ChatFlowAPI)
> Real-time chat backend with WebSocket support

- WebSocket connections via **SignalR** — rooms, live messaging, typing indicators, online presence
- JWT authentication through query string for SignalR hub
- Room roles: Admin / Moderator / Member
- Paginated message history, edit & delete; full unit test coverage

`ASP.NET Core 8` `SignalR` `SQL Server` `JWT` `xUnit` `Moq`

---

### 🤖 [Evolutionary Robots](https://github.com/softwacar/evolutionary-robots)

- Autonomous robots evolve over generations to collect energy and avoid obstacles
- Full GA pipeline: fitness scoring, elitism, single-point crossover, per-gene mutation across 4 DNA traits
- Interactive parameter panel + real-time fitness chart comparing 3 simulations

`Python` `Pygame` `Matplotlib` `NumPy` `Genetic Algorithms`
